# 텍스트 분석

## 텍스트 사전 준비(텍스트 전처리) - 텍스트 정규화


### 클렌징(Cleansing)

텍스트에서 분석에 오히려 방해가 되는 불필요한 문자, 기호 등을 사전에 제거하는 작업이다. <br> 예를 들어 HTML, XML 태그나 특정 기호 등을 사전에 제거한다.




### 텍스트 토큰화(Tokenization)

문서에서 문장을 분리하는 문장 토큰화와 문장에서 단얼르 토큰으로 분리하는 단어 토큰화로 나눌 수 있다.

- 문장 토큰화

- 단어 토큰화
